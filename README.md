# Pedra, Papel e Tesoura

Projecto de conclusão do módulo de Funções JavaScript do The Odin Project.
Jogo jogado inteiramente no console do browser.

## Como jogar

1. Clona este repositório ou faz o download dos ficheiros.
2. Abre o ficheiro `index.html` no teu navegador (Chrome, Firefox ou Edge).
3. Abre as **Ferramentas do Programador** (Pressiona `F12` ou `Ctrl+Shift+I`).
4. Clica na aba **Console**.
5. Segue as instruções no ecrã (introduz o teu nome e faz as tuas jogadas).
4. Escolhe entre `rock`, `paper` ou `scissors` em cada rodada
5. O jogo dura 5 rodadas — o jogador com mais pontos vence

## Tecnologias

- Html
- JavaScript
- Git

## Conceitos aplicados

* **Escopo de Variáveis**: Uso de variáveis globais e locais.
* **Funções**: Criação de funções com retorno e parâmetros.
* **Logica Condicionais**: Estruturas `if / else if / else` para lógica de decisão.
* **Manipulação de Strings**: Uso de `.toLowerCase()` para tratar entradas do utilizador.
